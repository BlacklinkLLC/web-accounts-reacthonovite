What's New in Accounts 3.9

Bridge release introducing teams, organizations, modular architecture, and 25+ premium themes.

  • Organizations & Teams with role-based permissions (owner, admin, member)
  • Shared AeroAI credits pool for team collaboration
  • 25+ premium themes including Dracula, Monokai, Tokyo Night, and more
  • Modular CSS & JavaScript architecture for better maintainability
  • ULTRA+ tier with priority features and beta access
  • Advanced analytics tracking system for user interactions
  • ClassLink discovery mode and Connect API groundwork
  • Enhanced SSO with JWT token support and external providers
  • Accessibility options (reduce motion, high contrast, font scaling)
  • Organization dashboard with member management
  • Live theme preview with interactive picker
  • Comprehensive Firestore security rules for organizations

    Connect & SSO Enhancements

    Major updates to authentication and rostering:

    • ClassLink Integration - OneRoster support for rostering data (classes, districts)
    • OpenSiS Framework - Powers upcoming SiS product (github.com/BlacklinkLLC/OpenSiS)
    • Third-Party SSO - Allow external apps to integrate Blacklink authentication
    • External SSO URL - accounts.blacklink.net/sso/external?token=[JWT]
    • Privacy Controls - User-controlled data sharing through SSO page

    Platform Technology

    Blacklink Accounts is built on modern, scalable Firebase infrastructure with security-first architecture.

  • Firebase Authentication - Multi-provider SSO support
  • Cloud Firestore - Real-time NoSQL database
  • Firebase Hosting - Global CDN delivery
  • Advanced Security Rules - Comprehensive access control
  • Custom CSS Variables - Dynamic theming system
  • Buy Me a Coffee API - Subscription payments
